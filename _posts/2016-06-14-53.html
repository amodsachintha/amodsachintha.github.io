Hello World කියන ටෙක්ස්ට් එකක් output කරමු.

තේරුම් ගත යුතු කොටස් කිහිපයක්.

 

#include<iostream> - Screen එකට Output කරන්න, Keyboard එකෙන් input ගන්න අපි iostream                                        (input output stream) හෙඩර් ෆයිල් එක භාවිතා කරනවා. ඉස්සරහට තව හෙඩර් ෆයිල්                                           වලින් වැඩ තියේ.

using namespace std; - cout, cin commands ඉදිරියෙන් std:: නොගහ කම්පයිල් කරන එක ලේසි                                                           නිසා using namespace std; යොදා ගන්නවා. (ලොකු කෝඩ් ලියන කොට                                                             namespaces වලට බෙදන්නයි මෙක use වෙන්නේ.)

int main() {  } - අපේ main ෆන්ෂන් (funtion) එක. අර curly brackets 2 ඇතුලේ තමා අපි කෝඩ්                                              ලියන්නේ. ඉස්සරහට main වලට අමතරව තව funtion වැඩ ගොඩක් එනවා.

return 0; - Program එකේ වැඩ ටික කිසි අවුලක් නැතිව උනා කියල OS එකට කියන්නේ මෙකෙන්.                                             මේවට "එරර් කෝඩ්ස්" Error codes කියල කියන්නේ. Error නැත්නම් return 0; ,                                               errors තියේ නම් return -1; මේවත් ඉදිරියේදි භාවිතා වේවි.

//comments - කෝඩ් එක තව කෙනෙක්ට තේරුම් ගන්න ලේසි වෙන්න අපි කොඩ් එක ඇතුලෙම                                                          කමෙන්ට් කරනවා. කමෙන්ට් පටන් ගන්නේ // න්. දිග රචනාවක් එහෙම කමෙන්ට්                                                        කරනවා නම්, /* ඔබේ රචනාව */
