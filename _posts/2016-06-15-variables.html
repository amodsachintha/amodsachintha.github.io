අපට අවශ්‍යය අගයන් පරිගණකයේ මතකයේ රදවා තබන්න ඒවා නැවත නැවත භාවිතයට ගන්න අපට variables පාවිච්චි කරන්න පුළුවන්.

C++ වල variables හදුන්වාදෙන්නේ (declare) කරන්නේ මෙන්න මේ විදියට:

int - දශමස්ථාන නැති අගයන්. උදා: 1, 35564, -45, -8.

float - දශමස්ථාන සහිත අගයන්. උදා: 0.22353, -5432.563

double - දශමස්ථාන සහිත, float වලට වඩා දශම ස්ථාන දරාගත හැකි type එකකි.
උදා: 5.000893313349

char - එක අකුරක් හෝ අංකයක්. උදා: a, v, 4, f

string - char එකතුවක්. උදා: I am 21 years old.

bool - ද්වීමය type එක. තිබිය හැකි අගයන් 2 කි. true , false
