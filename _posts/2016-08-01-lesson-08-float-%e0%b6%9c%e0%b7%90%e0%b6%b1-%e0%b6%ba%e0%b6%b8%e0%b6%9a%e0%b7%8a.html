C++ වල float ඩේටා ටයිප් එක if වගේ comparison operator එකකට යොදාගන්න විට මතක තබා ගත යුතු කාරණාවක් තියේ.

float value = 3.5;

ඉහත value වේරියබල් හි අගය 3.5ක් ලෙස දුන්නද එය පරිඝණකයේ මතකයේ රැදෙන්නේ 3.500000054 ලෙස.
එවිට   3.5    !=   3.500000054      (!=  <-- අසමානයි).
