කන්ඩිෂන් එකක් පිලිපදිමින් interactive output දෙන්න ඉෆ් else else if භාවිතා කරන්න පුලුවන්.

>     විශාලද?                12 > 10 is TRUE
<     කුඩාද?                 2 < 5 is TRUE
>=    විශාල හෝ සමානද?       4 >= 4 is TRUE
<=    කුඩා හෝ සමානද?        3 <= 4 is TRUE
==    සමානද?                33 == 33 is TRUE
!=    අසමානද?               11 != 4 is TRUE

මූලික syntax එක,

if ( TRUE ){
  මෙම statement එක execute කරන්න.
}
else if(TRUE){
  මේවා execute කරන්න.
}
else{
 මේවා execute කරන්න.
}
